# project-wm Development Guidelines

Main projects are called TerminalWindowManager.*
It is a combination of C#, Rust, and TypeScript projects with Tauri used for the desktop UI/runtime.
 - Primary UI project: TerminalWindowManager.Tauri
 - ConPTY helper: TerminalWindowManager.ConPTYHost
 - Shared parser/domain project: TerminalWindowManager.Core
